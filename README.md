# Powershell-Scripts

A collection of PowerShell scripts for IT administration and managed service provider (MSP) operations. Scripts are organized by platform or service category. Each script includes its own README with prerequisites, configuration details, and usage instructions.

---

## Repository Structure

```
Powershell-Scripts/
│
├── Microsoft_365/
│   └── Create_User_Password_TOTP_and_Assign_Lic/

```
---
## Usage

1. Clone or download the repository
2. Navigate to the script folder you need
3. Read the README for that script before running — each one documents its required variables, connection scopes, and any prerequisites
4. Connect to the relevant service (e.g. `Connect-MgGraph`) as documented
5. Run the script from PowerShell

---

## Notes

- All scripts are maintained for use in production MSP environments
- Scripts that interact with Microsoft Graph require appropriate admin roles in the target tenant — see individual READMEs for specifics
- Credentials or sensitive output generated by scripts should be stored in a password manager immediately and never committed to this repository
- This is a private repository — do not store tenant-specific configuration, credentials, or client data in any script files
